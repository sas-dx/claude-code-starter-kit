Claude Code Starter Kit

Claude Code 主要機能の日本語リファレンス & 実践ガイド

はじめに

Claude Code を初めて使う方は、まず導入ガイドをご覧ください。インストールから初回起動、基本操作までを新入社員でも迷わないレベルで解説しています。

🚀

導入ガイド

Node.js / Git の確認、Claude Code のインストール、認証、基本操作、IDE 連携、トラブルシューティングまで。

このサイトの使い方

📖

機能を学ぶ

左のサイドバーから各機能のページへ。仕組み・書き方・サンプルコードを網羅的に解説しています。

🏗️

実践ガイドで設計する

「実践ガイド」セクションでは、実際のプロジェクトを題材に全機能を組み合わせたアーキテクチャを解説。設定ファイル全文付き。

🔍

検索する

Ctrl+K でページ内検索。知りたいキーワードですぐに該当箇所にジャンプできます。

機能リファレンス

Claude Code が提供する各機能の詳細ページです。仕組み、設定方法、サンプルコードを解説しています。

📋

CLAUDE.md

セッション開始時に自動読み込みされるプロジェクト指示書。チーム全員が同じルールで作業でき、AI の暴走も防止します。
📏

Rules

CLAUDE.md の詳細版として機能するルールファイル。パス別ルールで特定ファイル操作時のみ適用させることも可能です。
⌨️

Commands

よく行う作業を /コマンド名 の一言で実行。コードレビュー、Issue ベース実装、コンフリクト解決などを提供。
🎯

Skills

コマンドの上位互換。手動呼び出しに加え、Claude が状況に応じて自動的に使用。テンプレートやスクリプトの同梱にも対応。
🤖

Agents

特定タスクに特化したサブエージェント。隔離コンテキストで独立動作し、Claude が必要に応じて自動的に委譲します。
👥

Agent Teams

複数の Claude Code インスタンスが並列で協調作業を行う実験的機能。共有タスクリストで自律的に協調します。
🪝

Hooks

ツール実行前後に自動実行されるシェルスクリプト。CLAUDE.md が「指示」ならば、Hooks は「仕組みで強制する」ガードレールです。
🔌

MCP

AI に GitHub、DB、Slack 等の外部サービスへのアクセスを提供する仕組み。Serena や Context7 を導入済み。
🧩

Plugins

Skills・Agents・Hooks・MCP サーバー等を1つのパッケージにまとめて配布・共有できる仕組みです。
⚙️

Settings

Claude Code の動作・権限・フック・環境変数を制御する設定ファイル。優先度付きの階層構造で管理します。
⌨️

Keybindings

キーボードショートカットのカスタマイズ。/keybindings で設定ファイルを開いて編集できます。

実践ガイド

Claude Code の各機能を組み合わせて、実際のプロジェクトをどう構築するかを具体的に解説します。設定ファイル全文・Mermaid 図付き。

🧠

エージェントアーキテクト入門

AIエージェントを設計する上で必要な5つの観点を解説。トークン最適化、品質向上、外部連携、継続学習、チーム展開。
🔧

ハーネスエンジニアリング

Claude Code の6つの拡張ポイントを体系的に組み合わせ、確率的AIを決定的制御で囲む設計手法を解説します。
🏗️

AI駆動開発アーキテクチャ

基幹システムリプレイスを題材に、Claude Code の全機能を組み合わせたチーム開発のグランドデザインを解説します。
📢

AI駆動マーケティング基盤

コンテンツ生成、SEO最適化、キャンペーン管理をClaude Codeで自動化するマーケティング基盤の構築ガイド。
💼

AI駆動セールス基盤

提案書自動生成、CRM連携、商談分析をClaude Codeで支援するセールス基盤の構築ガイド。
📊

AI駆動データドリブン基盤

ETL、レポート自動生成、データ品質監視をClaude Codeで構築するデータ分析基盤ガイド。
